# tetrads
<pre>
find tetrads
Compute tetrads, figures made of four congruent shapes, so that each shares a boundary with all others.
see https://userpages.monmouth.com/~colonel/tetrads/ntetrads.html for details & solutions
Ex : 
shape =
      x x
        x
        x
        x
      x x x

tetrad =
         2
 2 2 2 2 2
 2 1 1 1 2
 3 3 1 4 4
   3 1   4
   3 1   4
   3 1 1 4
 3 3 3 4 4 4
 
 1 touches 2,3,4
 2 touches 1,3,4
 3 touches 1,2,4
 4 touches 1,2,3
 </pre>
